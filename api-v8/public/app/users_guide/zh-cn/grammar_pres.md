# 现在式

**动词**变位之一，表示当下发生的动作。
其中，动词的现在式的第三人称单数变化，作为词典收录动词的标准形式——**词典源型**。

|语种|缩写|全称|
|-|-|-|
|巴利|——|paccuppanna|
|汉|现|现在式|
|英|pres.|present tense|
